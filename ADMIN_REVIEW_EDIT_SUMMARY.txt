SYSTEM ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

✓ Original Reviews (localStorage via ReviewsContext)
  - Customer reviews stored here
  - Used by home page and products
  - Unchanged by admin edits

✓ Admin Edits (sessionStorage via AdminReviewsStorageContext)
  - Admin modifications stored separately
  - Session-based (clears on logout)
  - Only visible in admin panel

✓ No Data Sync Between Systems
  - Complete isolation
  - Zero risk of data corruption
  - Home page always shows original reviews

HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. Login to admin panel (/admin) with numeric PIN
2. Go to "Reviews & Ratings" tab
3. Click blue "Edit" button on any review
4. Edit title, rating, or comment in modal
5. Click "Save" to store admin edit
6. Review shows blue badge: "ADMIN EDITED"
7. Original review on homepage remains unchanged
8. On logout, all session edits are cleared
