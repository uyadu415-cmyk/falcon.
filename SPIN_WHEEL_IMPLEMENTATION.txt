✅ 4. SPIN WHEEL REWARDS (Requirement Met)

Segments:
  1. 10% Discount     → Weight: 50%
  2. Better Luck      → Weight: 35%
  3. 20% Discount     → Weight: 10%
  4. 30% Discount     → Weight: 5%
  5. 40% Discount     → Weight: 0% (Display only)
  6. 50% Discount     → Weight: 0% (Display only)

Probability System:
  const WEIGHTS = [50, 35, 10, 5, 0, 0]
  const sum = WEIGHTS.reduce((a, b) => a + b, 0)  // = 100
  
  let r = Math.random() * sum
  for (let i = 0; i < WEIGHTS.length; i++) {
    r -= WEIGHTS[i]
    if (r <= 0) return SEGMENTS[i].discount
  }

Result: User can only win 10%, 20%, 30%, or "Better Luck"
        User CANNOT win 40% or 50% (impossible)
