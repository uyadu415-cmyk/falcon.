## 🛠️ Setup Required

### Before Deployment

1. **Vercel Account** (Free tier works)
   - https://vercel.com/signup

2. **GitHub Account** (For easy deployment)
   - https://github.com/signup

3. **Environment Variables** (Configure in Vercel)
   ```
   ADMIN_PASSWORD = your-secure-password
   OWNER_EMAIL = your-email@gmail.com
   ```

4. **Vercel Blob** (Auto-configured via integration)
   - For storing products and images

5. **Optional: Resend** (For email notifications)
   - https://resend.com
